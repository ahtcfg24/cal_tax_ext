# cal_tax_ext
一个用来计算税后薪资的chrome插件

# 税后收入计算器（Chrome 扩展）使用说明

## 安装
1. 下载或克隆项目代码。
2. 在 Chrome 打开 `chrome://extensions/`。
3. 右上角开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择项目下的 `chrome-extension` 目录。

## 使用步骤
1. 在弹窗顶部选择工作城市；如选择“其他城市”，可自定义养老/医疗/失业个人比例（默认 8%、2%、0.5%，基数不限）。
2. 依次填写：
   - 税前月薪
   - 社保缴费基数 / 公积金缴费基数（城市支持自动上下限；“其他城市”无限制）
   - 公积金比例（个人/公司，默认 12%）
   - 专项附加扣除（可勾选快速项，自动累加；互斥项会自动取消）
   - 年终奖金额、发放月份、计税方式（单独计税 / 累计并入）、快速倍数填充
3. 点击“计算税后收入”，向下滚动查看结果。

## 结果说明
- 月度核心数据：税前、社保及公积金、个税、税后、含公积金税后（含年终奖时标记）。
- 月度详细明细：分项展示养老/医疗/失业比例与金额、公积金个人/公司、累计减除费用、当月个税等。
- 年度汇总：年度税前总收入、年度个人缴纳社保、年度个人缴纳公积金、年度缴纳个税、年度税后总收入、年度含公积金税后总收入。

## 常见配置
- 城市切换：自动应用各城市社保/公积金基数上下限及个人比例；医疗固定额（如 2%+3）自动识别并参与计算与文案。
- 其他城市：不限基数，自定义三项比例，医疗固定额默认为 0。
